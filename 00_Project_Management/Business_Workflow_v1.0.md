# Project Nova - Business Workflow v1.0

## Primary Actor

Client (Employer / HR)

---

## Workflow

1. Client logs in
2. Client creates a verification request
3. Client enters candidate details
4. Candidate receives verification request
5. Candidate uploads required documents
6. System validates documents
7. APIs perform digital verification
8. Manual verification starts where required
9. Verification checks complete
10. AI Risk Engine generates insights
11. Final report is generated
12. Client downloads report
13. Audit log is recorded
14. Billing updated
15. Notification sent

---

- Authentication
